# Housing Price Prediction (Ames Housing Dataset)

This project predicts house prices using the Ames Housing dataset. The notebook includes EDA, feature engineering, model building, and evaluation.

## Files in this Repository
- `Housing_big_data_GitHub_Ready.ipynb` – Final cleaned notebook
- `ames iowa housing.csv` – Dataset used in this project
- `requirements.txt` – Python dependencies

## Libraries Used
- pandas
- numpy
- seaborn
- matplotlib
- scikit-learn

## What the Notebook Includes
- Data loading
- Missing value checks
- Data types checks
- EDA and visualizations
- Feature engineering
- Train-test split
- Model building
- R² and RMSE evaluation
